# Телеграм бот

## В проекте используются такие библиотеки как:

    aiogram==2.24
    face-recognition==1.3.0
    pydub==0.25.1

### Перед работой требуется установить зависимости из файла requirements.txt

```shell
pip install -r requirements.txt
```

### Запуск бота

```shell
python main.py
```
